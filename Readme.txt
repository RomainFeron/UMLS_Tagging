## General:
UMLS tagging tool by Issa Diop, Romain Feron and Wilfried Poudroux.
Version 1.0 (2015-12-06)
Find out more on github: http://github.com/RomainFeron/UMLS_tagging
-------------------------------------------------------------------
## Requirements:
- Python 3.4 or higher
- UMLS metathesaurus (2011)
- SFBI mails archives
-------------------------------------------------------------------
## How to use:
- Place the metathesaurus in the "ressources" folder
- Place the mails archives in "ressources/mail" folder
- Check the path and name of files in "umls_tagger.py"
- Set parameter (conservative or not)
- Run "umls_tagger.py"
- Be patient
